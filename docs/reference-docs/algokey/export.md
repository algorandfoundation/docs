## algokey export



Export key file to mnemonic and public key



### Synopsis



Export key file to mnemonic and public key



```

algokey export [flags]

```



### Options



```

  -h, --help                help for export

  -f, --keyfile string      Private key filename

  -p, --pubkeyfile string   Public key filename

```
